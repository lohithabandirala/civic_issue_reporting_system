# Vive Le Tech Admin Panel

Admin dashboard for managing Vive Le Tech event registrations and participants.

## Features
- Admin Login
- Dashboard Analytics
- Event Registrations Management
- Participants Management
- Responsive Sidebar Navigation

## Tech Stack
- React.js
- JavaScript
- Firebase
- Tailwind CSS
- Vercel Deployment
